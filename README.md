# TGgameeCheat

Gamee telegram robot games cheat

# Installation

```
git clone https://github.com/B4rC0d/TGgameeCheat.git
```
Go to the directory
```
cd TGgameeCheat
```
Install NodeJS(arch)
```
sudo pacman -S nodejs
```
Install NodeJS(debian)
```
sudo apt install nodejs
```
Installing Node JS library
```
npm install md5
```
Run
```
python3 TGgameeCheat.py
```
